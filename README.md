## ‚óº The Zero-Trace Protocol (Read First)

**Quorix is the world's first decentralized meeting environment designed to operate without client-side JavaScript execution.**

In an era of ubiquitous digital surveillance, standard web-based meeting platforms fingerprint your device, track your IP, and leak metadata through complex script execution. Quorix eliminates this vector entirely.

By running locally as a raw HTML/CSS extension, **we strip away the browser's ability to fingerprint your machine.**

## ‚óº Deployment Protocol (Installation)

To maintain the **Zero-JavaScript** security environment, Quorix must be loaded directly into the browser kernel as a local extension. This prevents third-party interference.

**Every participant must follow these steps once:**

1.  **Download** the latest release `quorix-release.zip` file from this repository.
2.  Open Google Chrome (or Brave/Edge) and navigate to: `chrome://extensions`
3.  Toggle **"Developer mode"** in the top right corner.
4.  Drag the `quorix-release.zip` file from your files app to the webpage.

*Once loaded, you will be redirected to a secure and isolated QUORIX- webpage, enjoy!.*
